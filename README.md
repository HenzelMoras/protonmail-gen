# protonmail-gen
Automate  proton mail  creation for various uses

### Steps
```bash
git clone https://github.com/HenzelMoras/protonmail-gen.git
cd protonmail-gen
```

### setup installation 
```bash
chmod u+x setup.py
python3 setup.py
```

### Usage
```bash
chmod u+x email-genbot.py
python3 email-genbot.py
```
